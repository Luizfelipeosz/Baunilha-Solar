# 📌 Visão Geral

Baunilha Solar é uma aplicação desenvolvida para simular fluxos comuns encontrados em plataformas reais de e-commerce.

O projeto foi estruturado com foco em organização, previsibilidade e evolução contínua da aplicação.

Entre os principais objetivos estão:

* Construção de interfaces responsivas
* Organização modular de estilos e scripts
* Persistência de dados no cliente
* Separação entre interface e comportamento
* Estrutura preparada para futura integração com backend

# 🎨 Organização do CSS

A arquitetura de estilos foi separada por contexto de página para reduzir acoplamento e melhorar manutenção.

### Estratégias utilizadas

* CSS modularizado por páginas
* Responsividade mobile-first
* Padronização visual reutilizável
* Separação entre estrutura e estilização

### Objetivos da estrutura

* melhorar legibilidade
* facilitar manutenção
* garantir consistência visual
* preparar a aplicação para crescimento futuro

---

# ⚙️ Decisões Técnicas

Algumas decisões adotadas durante o desenvolvimento:

### Arquitetura multipágina

Separação de responsabilidades entre páginas para simplificar manutenção e organização.

### Persistência utilizando localStorage

Dados mantidos no navegador para simular comportamento de aplicações reais.

### Organização modular

Scripts e estilos separados por contexto para melhorar previsibilidade e escalabilidade.

### Estrutura preparada para backend

Aplicação organizada pensando em futura integração com APIs e autenticação real.
